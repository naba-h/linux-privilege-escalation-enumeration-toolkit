🧠 Overview

Privilege escalation is one of the most critical stages in a security assessment.
Once access is obtained, elevating privileges determines the real impact.

This toolkit automates enumeration by detecting:

- Misconfigured permissions
- Insecure files
- Weak configurations
- Hidden escalation vectors

«⚡ Built for speed, clarity, and real-world exploitation workflows»

---

⚡ Features

- 🔍 Automated privilege escalation checks
- 📁 Misconfiguration detection
- ⚠️ SUID/SGID analysis
- ⚡ Lightweight & fast
- 🧠 Clean structured output
- 🛠 Easy to customize
